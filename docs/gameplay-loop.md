## 목적
플레이 루프를 단계별로 정의하여 시스템 구현 순서와 밸런싱 기준을 제공합니다.

## 범위
메인 루프와 단계별 목표/해금/실패 위험을 다룹니다.

## 본문
### 메인 루프
`수동 채굴 -> 기초 생산 -> 자동화 구축 -> 생산 확장 -> 고급 자원 확보 -> 점프 드라이브 제작 -> 행성 코어 채굴 -> 탈출`

### 단계별 루프 상세
| 단계 | 플레이 목표 | 핵심 시스템 | 다음 단계 진입 조건 |
| --- | --- | --- | --- |
| 1단계: 생존 기반 | 기본 자원 확보 | 수동 채굴, 기본 제작 | 자동 채굴기 제작 가능 |
| 2단계: 자동화 시작 | 반복 작업 자동화 | 채굴기, 컨베이어, 기본 전력 | 병목 구간 1차 해결 |
| 3단계: 생산 확장 | 중급 자원 체인 구성 | 조립 라인, 물류 분기 | 고급 재료 생산 가능 |
| 4단계: 탈출 준비 | 최종 장치 준비 | 점프 드라이브 부품 생산 | 코어 채굴 장비 준비 |
| 5단계: 종결 | 탈출 실행 | 코어 채굴, 점프 드라이브 작동 | 엔딩/타임랩스 |

### 설계 포인트
- 각 단계는 새 시스템 도입과 기존 병목 해결을 동시에 요구해야 합니다.
- 기술 해금은 루프 전환의 명확한 신호로 동작해야 합니다.
- 엔딩은 생산 체인의 누적 결과를 보여주는 방식으로 마무리합니다.

### 문서 수정 트리거
- 루프 단계 수 또는 단계명 변경
- 단계 진입 조건 또는 목표 변경
- 기술 해금/엔딩 연출 설계 변경

## 결정사항
- 플레이 루프는 5단계 구조로 정의합니다.
- 루프 전환 기준은 "기술/생산 병목 해결"로 통일합니다.
