# _Project 폴더 스캐폴딩

이 디렉터리는 `docs/development/project-structure.md`의 권장 구조를 기준으로 생성한
초기 Unity 자산 폴더 뼈대입니다.

## 포함된 상위 구조
- `Art/`, `Audio/`, `Materials/`, `Prefabs/`, `Scenes/`
- `Scripts/Runtime/{Core,Gameplay,Systems}`
- `Scripts/Editor/`, `Scripts/Tests/`
- `Data/{Resources,Buildings,Tech}`

빈 디렉터리 보존을 위해 각 폴더에 `.gitkeep` 파일을 둡니다.
